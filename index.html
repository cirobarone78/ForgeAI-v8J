      🚀 Guida al Deploy Fullstack
      
    

    

      

        Forge ha generato il tuo progetto fullstack con frontend e backend separati.
        Segui questi step per renderlo live in ~5 minuti.
      

      


        

          

            
1

            

              
📥 Scarica e struttura i file

              
Organizza frontend e backend in cartelle separate

            

          

          

            Clicca ↓ Esporta per scaricare tutti i file. Poi organizzali così:
            
mia-app/
├── frontend/
│   └── index.html
└── backend/
    ├── server.py (o server.js)
    └── requirements.txt (o package.json)

            Poi carica su GitHub con il pulsante 🐙 GitHub.

              Su Railway:

              1. Vai su railway.app → New Project

              2. Deploy from GitHub → seleziona la tua repo

              3. Railway rileva automaticamente Python/Node

              4. Copia l'URL del backend generato (es. https://mia-app.railway.app)
            

          

        


        

          

            
3

            

              
🌐 Deploy Frontend

              
GitHub Pages (già attivato) o Netlify

            

          

          

            Se hai già usato il pulsante GitHub, Pages è attivo automaticamente.


            Aggiorna l'URL del backend nel frontend:

            Nel file index.html, sostituisci localhost:5000 con il tuo URL Railway:

            
const API = 'https://mia-app.railway.app';

            Poi fai un nuovo push su GitHub — Pages si aggiorna in automatico.
